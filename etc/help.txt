Зарезервированные имена:

01. Program - начало программы;
02. begin   - начало алгоритмического блока;
03. end     - завершение алгоритмического блока;
04. var     - начало блока описания переменных;
05. integer - тип переменных для целых чисел;
06. real    - тип переменных для вещественных (дробных) чисел;
07. write()/writeln() - отображение текста без/с переходом на другую строку;
08. read()/readln()   - ввод данных без/с переходом на другую строку;
09. trunc() - возвращает целую часть числа;
10. round() - возвращает ближайшее целое число;
11. frac()  - возвращает дробную часть числа;
12. abs()   - возвращает модуль числа;
13. sqr()   - возвращает квадрат числа;
14. sqrt()  - возвращает корень числа (только для чисел >0);
15. (переменная) := (значение) - присвоить значение переменной;
15. + - / * - операции с числами (сложить/отнять/делить/умножить);
16. div     - целочисленное деление одного числа на другое (a div b);
17. mod     - деление по модулю (возвращает остаток от деления) (a mod b);
18. = - равно; <> - не равно; >= больше или равно; <= меньше или равно;
19. and - логическое И; or - логическое ИЛИ; xor - логическое исключающее ИЛИ;
20. if (условие) then (действие если условие верно) else (действие если условие не верно)
21. for i:=(повторять от) to (повторять до) do (что вообще повторять пока i<>повторять до)
22. while (условие исполняется) do (выполняем действие)
23. repeat (действие) until (пока условие не выполнится)
